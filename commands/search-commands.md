# 搜索口令（粘进你 AI 的常驻记忆，如 Claude Code 的 CLAUDE.md）

把下面这段写进你 AI Agent 的持久记忆，任何会话开场即生效。之后你只说人话 + 前缀，AI 自动选路、并行检索、合并成带来源的小报告——**全程无头后台、不打扰**。

```
### 搜索口令
- `ss <内容>`  = 全平台（中国组 + 外国组），合并成带来源的报告
- `ssc <内容>` = 全中国（元宝 + 小红书 + B站 + 知乎 + V2EX；走量加 TikHub）
- `ssf <内容>` = 全外国（推特/X + Reddit + WebSearch + YouTube；技术加 HackerNews/Arxiv）

执行细则：
- 登录类平台走「专用无头浏览器」（独立内核，绝不碰用户日常 Chrome）。
- 多平台并行；人速限流（小红书尤其：间隔几秒、单次几十条内，见验证码即停）。
- 输出 = 按平台分块 + 一段综合结论 + 来源链接 的小报告。
- 中文实时/公众号一律带元宝；关键结论可用 元宝 / Kimi / DeepSeek / 秘塔 多源交叉核实。
- 口令仅作用于「信息检索」，不触发发帖/评论/点赞等写操作。
```
